# servi-os.profissionais-
# 🚀 Portfolio de Serviços Profissionais - Thayanne Oliveira

Este é o repositório do meu site profissional de serviços, focado em elevar o posicionamento de carreira de estudantes e profissionais da área tech. O projeto combina uma estética moderna (Dark Mode & Glassmorphism) com uma arquitetura front-end limpa e responsiva.

📍 **Acesse o site:** 

---

## ✨ Funcionalidades

- **Design Premium:** Estética *Dark Mode* com detalhes em *Midnight Pink* e efeitos de iluminação dinâmica.
- **Glassmorphism:** Cards de serviços com efeito de vidro fosco utilizando `backdrop-filter`.
- **Animações Fluidas:** Integração com a biblioteca *ScrollReveal* para uma experiência de navegação suave.
- **Responsividade Total:** Interface adaptável para dispositivos móveis, tablets e desktops.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** - Variáveis CSS para fácil manutenção.
  - Flexbox e CSS Grid para layout.
  - Animações `@keyframes` para elementos flutuantes (Blobs).
- **JavaScript:** - Manipulação de DOM para interatividade.
  - ScrollReveal.js para animações de entrada.
- **Fontes:** Poppins & Playfair Display (Google Fonts).
